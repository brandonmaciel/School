The following information is based on Mac.

To run code, navigate to the directory of the source code then run:

        python3 parser.py FILE_NAME.txt
    
    
We have included 2 demo text files, one that runs normally and one that generates an error.
To use run these example use:

        python3 parser.py demo.txt

for a normal run, and

        python3 parser.py demo_error.txt

for an example that generates an error.
